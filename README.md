## 🚀 프로젝트 소개
클릭투어는 나만의 여행 플래너를 제작할 수 있는 웹사이트입니다.<br/>카카오맵API를 사용하여 여행 플래너를 제작할 수 있습니다. <br/>다른 사람들의 플래너를 구경하고 나만의 플래너로 커스텀 시킬 수도 있습니다!

## 📝 프로젝트 기능
- 여행 플래너 제작
- 여행 플래너 랜덤 추천
- 플래너 복사하기
- 여행 플래너에 대한 후기 게시판
